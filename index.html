Введение в Bootstrap

Шаг 1. Введение в Bootstrap

Bootstrap - это «самая популярная в мире среда HTML, CSS и JS для создания адаптивных мобильных проектов в сети Интернете».

Преймущества Bootstrap

2

Bootstrap - это топ фреймворк, который в основном используются внешними разработчиками по разным причинам.

Визуальный Breakpoints

3

Bootstrap предоставляет различные обозначения на основе категорий и определяет классы CSS на основе них.

Структура 12 столбцов

4

Bootstrap обеспечивает 12-колоночную структуру сетки, чтобы сделать ее адаптивной. Bootstrap делит всю ширину устройства на 12 столбцов.

Добавление навигации

3

Bootstrap предоставляет компонент Navbar для отображения элемента навигации.

Компонент навигационной панели Bootstrap содержит кнопку-переключатель для отображения и скрытия элементов навигации.

Добавление Glyphicons

4

Glyphicon - это библиотека значков, которая использует символы Юникода для отображения значка.

Добавление панели

5

Панель состоит из 3 частей:

  • Заголовок
  • Тело - содержит основной контент.
  • Подвал.

Добавление Jumbotron

6

Jumbotron используется для отображения ведущего текста на веб-странице. Мы можем использовать jumbotron, чтобы выделить некоторую информацию на странице.

Добавление компонентов

Шаг 3. Добавление компонентов

Контент сайта и качество наполнения сайта напрямую отвечают за успех интернет-проекта.

Добавление отзывчивого
изображения

1

Обычно изображения отображаются как встроенный элемент в браузере. Так что bootstrap предоставляет класс .img-отзывчивый для создания адаптивных изображений.

Добавление отзывчивой таблицы

2

Таблицы используются для отображения структурированных данных в виде строк и столбцов. Boostrap предоставляет класс .table для создания адаптивной таблицы.

Добавление отзывчивого видео

3

Bootstrap предоставляет классы, чтобы сделать внедрение адаптивным. Большую часть времени нам требуется отображать сторонние приложения на нашей веб-странице.
