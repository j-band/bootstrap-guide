Введение в Bootstrap

Шаг 1. Введение в Bootstrap

Bootstrap - это самая популярная в мире среда HTML, CSS и JS для создания адаптивных мобильных проектов в сети Интернете.

Адаптивный дизайн

1

Концепция дизайна

Преймущества Bootstrap

2

Bootstrap - это топ фреймворк, который в основном используются Front-end разработчиками с самыми разными целями.

Визуальный Breakpoints

3

Bootstrap предоставляет различные обозначения и определяет классы CSS на основе них. Breakpoints сетки основаны на медиазапросах минимальной ширины.

Структура 12 столбцов

4

Bootstrap обеспечивает 12-колоночную структуру сетки, чтобы сделать ее адаптивной. Bootstrap делит всю ширину устройства на 12 столбцов.

Добавление Font Awesome

Font Awesome - это набор шрифтов и иконок. Он был создан для использования с Bootstrap и включен в BootstrapCDN.

Шаг 3. Добавление контента

Контент сайта и качество наполнения сайта напрямую отвечают за успех интернет-проекта.

Добавление отзывчивого
изображения

1

Bootstrap предоставляет класс .img-fluid для создания адаптивных изображений. Этот класс делает их адаптивными для устройств разных размеров.

Добавление отзывчивой таблицы

2

Таблицы используются для отображения структурированных данных в виде строк и столбцов. Bootstrap предоставляет класс .table для создания адаптивной таблицы.
