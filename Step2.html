Шаг 2. Создание каркаса

1. Создание каркаса

Каркасом - это основные блоки, которые разделяют контент сайта друг от друга. Самый простой и базовый пример: шапка – блок слева – блок справа и подвал (футер). В этих блоках в свою очередь может быть очень много других, вложенных элементов, но они уже являются дополнительной разметкой, которая добавляется на следующем этапе.

2. Добавление Bootstrap в проект

Мы будем создавать наш сайт с помощью Web Storm. Вы же можете выбрать любой редактор на ваш выбор.

Если вы хотите скачать и добавить Bootstrap самостоятельно, вы можете перейдите на официальный сайт Bootstrap и скачать его там.

Если вы не хотите загружать Bootstrap самостоятельно, вы можете включить его из CDN (Content Delivery Network). MaxCDN обеспечивает поддержку CDN для CSS и JavaScript Bootstrap. Вы также можете включить jQuery.

<link rel="stylesheet"
href="https://maxcdn.bootstrapcdn.com/bootstrap/4.0.0/css/bootstrap.min.css"

crossorigin="anonymous">

<!-- jQuery -->
<script src="https://code.jquery.com/jquery-3.4.1.slim.min.js"

3. Создание базового макета

В первую очередь, нам необходимо создать новую HTML-страницу index.html, после создания добавляем следующий код, это как раз и есть базовая структура.

Так же, не забудьте подключить bootstrap.css в head страницы.

А так же добавить ссылки на jQuery-3.5.0.js и bootstrap.js внизу тега body. Обычно загружают скрипт внизу страницы, чтобы улучшить время загрузки страницы, так как будет лучше, если HTML и CSS загрузяться первыми. 

    
<!DOCTYPE html>
<html>
     <head>
         <title>Home</title>
         <meta charset="utf-8" />
         <meta name="viewport" content="width=device-width,
         initial-scale=1, maximum-scale=1, user-scalable=no">
        <link rel="stylesheet"
href="https://maxcdn.bootstrapcdn.com/bootstrap/4.0.0/css/bootstrap.min.css"
integrity="sha384-Gn5384xqQ1aoWXA+058RXPxPg6fy4IWvTNh0E263XmFcJlSAwiGgFAW/dAiS6JXm"
crossorigin="anonymous">
    </head>
<body>
    <header class="container">
        <h2>Заголовок</h2>
    </header>
    <section class="container" >
        <div class="row">
          <div class="col-3 Nav">
          Навигация
          </div>
          <div class="col-6 content">
          Область содержимого
          </div>
          <div class="col-3 reletedContent">
          Контент
          </div>
        </div>
    </section>
    <footer class="container">
        <h2>Подвал</h2>
    </footer>

<script src="https://code.jquery.com/jquery-3.4.1.slim.min.js"
integrity="sha384-J6qa4849blE2+poT4WnyKhv5vZF5SrPo0iEjwBvKU7imGFAV0wwj1yYfoRSJoZ+n"
crossorigin="anonymous"></script>

<script src="https://cdn.jsdelivr.net/npm/popper.js@1.16.0/dist/umd/popper.min.js"
integrity="sha384-Q6E9RHvbIyZFJoft+2mJbHaEWldlvI9IOYy5n3zV9zzTtmI3UksdQRVvoxMfooAo"
crossorigin="anonymous"></script>

<script src="https://stackpath.bootstrapcdn.com/bootstrap/4.4.1/js/bootstrap.min.js"
integrity="sha384-wfSDF2E50Y2D1uUdj0O3uMBJnjuUD4Ih7YwaYd1iqfktj0Uod8GCExl3Og8ifwB6"
crossorigin="anonymous"></script>
</body>
</html>

Наш код создает разделы <header> <footer> и основной контейнер. Мы добавили класс .container к заголовку, всему содержимому и подвалу.

Внутри раздела добавили <div>с классом .row

Внутри класса .row добавили 3 класса столбцов для навигации и всего контента. Также определены классы .col-3, col-6, col-3 соответственно. Эти классы разделят общую структуру столбцов в соотношении 3:6:3.

Таким образом, навигация займет первые 3 столбца, область основного контента займет следующие 6 столбцов, а остальное содержимое займет последние 3 столбца.

4. Добавление навигации

Bootstrap предоставляет компонент Navbar для отображения элемента навигации. Компонент навигационной панели Bootstrap содержит кнопку-переключатель (3 маленькие горизонтальные линии справа) для отображения и скрытия элементов навигации. 

    
<nav class="navbar navbar-expand-lg navbar-light bg-light">
  <a class="navbar-brand" href="#">Navbar</a>
  <button class="navbar-toggler" type="button" data-toggle="collapse"data-target="#navbarNav"
        aria-controls="navbarNav" aria-expanded="false" aria-label="Toggle navigation">
    <span class="navbar-toggler-icon"></span>
  </button>
  <div class="collapse navbar-collapse" id="navbarNav">
    <ul class="navbar-nav">
      <li class="nav-item active">
        <a class="nav-link" href="#">Домой<span class="sr-only">(current)</span></a>
      </li>
      <li class="nav-item">
        <a class="nav-link" href="#">О нас</a>
      </li>
      <li class="nav-item">
        <a class="nav-link" href="#">Продукты</a>
      </li>
      <li class="nav-item">
        <a class="nav-link" href="#">Контакты</a>
      </li>
    </ul>
  </div>
</nav>

See the Pen gOaWWWO by Julia Kalinichenko (@yuliakalina) on CodePen.

В данном примере, использовано несколько классов для .navbar

  • .navbar-toggler для использования с нашим JS-«плагином свертывания» и других изменяющихся состояний навигации.
  • Навбар требуют «обертки» из классов .navbar с .navbar-expand {-sm | -md | -lg | -xl} для адаптивных классов свертывания.
  • Элемент </div> с .collapse.navbar-collapse для группирования и скрытия содержимого навбара на определенном брейкпойнте родителя.

4. Добавление Font Awesome

Font Awesome - это набор шрифтов и иконок, основанный на CSS и LESS. Он был создан для использования с Bootstrap, а позже был включен в BootstrapCDN.

После того, как вы приступите к работе, вы можете разместить иконки Font Awesome практически в любом месте, используя префикс fa и имя иконки. Font Awesome предназначен для использования со встроенными элементами (для краткости используться тег <i> (i -icon), но использование <span> более семантически правильно). 

    
<i class="fa fa-heart fa-lg"></i>
<i class="fa fa-heart fa-2x"></i>
<i class="fa fa-heart fa-3x"></i>
<i class="fa fa-heart fa-4x"></i>

Приведенные строки создадут значки сердца разных размеров. Полный список классов значков доступен здесь.

Если вы измените размер шрифта контейнера иконки, иконка станет больше. То же самое касается цвета, тени и всего, что наследуется с помощью CSS.

Вы можете поворачивать, анимировать, добавлять границы, растягивать, заменять маркеры.

  • Чтобы произвольно вращать и переворачивать значки, используйте классы fa-rotate- * и fa-flip- *.
  • Используйте класс fa-spin, чтобы повернуть любой значок, и используйте fa-pulse, чтобы он вращался с 8 шагами. Хорошо работает с fa-spinner, fa-refresh и fa-cog.
  • Используйте fa-border и fa-pull-right или fa-pull-left для простых цитат или значков статей.
  • Используйте fa-ul и fa-li для простой замены маркеров по умолчанию в неупорядоченных списках.
  • Используйте fa-fw, чтобы установить значки с фиксированной шириной. Отлично подходит для использования при разной ширине иконки. Особенно полезно в таких вещах, как навигационные списки и группы списков. 
    
<div class="list-group">
  <a class="list-group-item" href="#"><i class="fa fa-user fa-fw" aria-hidden="true"></i> О нас</a>
  <a class="list-group-item" href="#"><i class="fa fa-download fa-fw" aria-hidden="true">
</i> Скачать</a>
</div>


<div class="fa-3x">
    <i class="fas fa-spinner fa-pulse"></i>
    <i class="fas fa-cog fa-spin"></i>
    <i class="fas fa-spinner fa-spin"></i>
</div>

5. Добавление панели

Панели являются элементами контейнера и используются для хранения другого содержимого. Иногда нам нужно отобразить некоторый контент внутри окна, тогда мы можем использовать панели. Bootstrap предоставляет отзывчивые панели.

Панель состоит из 3 частей

  • Заголовок - определение заголовка панели.
  • Тело - содержит основной контент.
  • Подвал (footer) 
    
<div class="container">
  <h2>Панели с классом панели по умолчанию</h2>
  <div class="panel-group">
    <div class="panel panel-default">
      <div class="panel-heading">Панель с классом панели по умолчанию</div>
      <div class="panel-heading">Панель Заголовок</div>
      <div class="panel-body">Панель Контент</div>
    </div>
      <div class="panel-footer">Панель Подвал</div>
<div class="panel panel-primary">
      <div class="panel-heading">Панель главного класса</div>
      <div class="panel-body">Панель Контент</div>
    </div>
  • Используйте класс .panel-heading для создания заголовка панели.
  • Используйте класс .panel-body для создания тела панели.
  • Используйте класс .panel-footer для создания панели подвала.
  • Чтобы сгруппировать много панелей вместе, оберните </div>классом .panel-group вокруг них.

Чтобы раскрасить панель, используйте контекстные классы .panel-default, .panel-primary, .panel-success, .panel-info, .panel-warning или .panel-danger
