Шаг 2. Создание каркаса

1. Создание каркаса

Под каркасом понимаются самые основные блоки, которые разделяют содержимое друг от друга. Самый простой пример: шапка – блок слева – блок справа и подвал (футер). В этих блоках в свою очередь может быть очень много других, вложенных элементов, но они уже являются дополнительной разметкой, которая добавляется на следующем этапе.

2. Добавление Bootstrap в проект

Мы будем создавать наш сайт с помощью Web Storm. Если у вас нет Web Storm, вы можете выбрать любой редактор на ваш выбор.

Если вы хотите скачать и вставить Bootstrap самостоятельно, вы можете перейдите на официальный сайт Bootstrap и скачать его там.

Если вы не хотите загружать Bootstrap самостоятельно, вы можете включить его из CDN (Content Delivery Network). MaxCDN обеспечивает поддержку CDN для CSS и JavaScript Bootstrap. Вы также можете включить jQuery.

3. Создание базового макета

В первую очередь, нам необходимо создать новую HTML-страницу index.html, после создания добавляем следующий код.

Так же, не забудьте подключить bootstrap.css в head страницы.

А так же добавить ссылки на jQuery-3.5.0.js и bootstrap.js внизу тега body. Обычно загружают скрипт внизу страницы, чтобы улучшить время загрузки страницы, так как нам нужно, чтобы HTML и CSS загружались первыми.

    </head>
<body>
    <header class="container">

Эта строка обеспечивает правильное отображение веб-страницы на мобильных устройствах. 

    
<meta name="viewport" content="width=device-width,
initial-scale=1, maximum-scale=1, user-scalable=no">

Наш код создает разделы <header> <footer> и основной контейнер. Мы добавили класс .container к заголовку, всему содержимому и подвалу.

Внутри раздела добавили <div>с классом .row

Внутри класса .row добавили 3 класса столбцов для навигации и всего контента. Также определены классы .col-md-3, col-md-6, col-md-3 соответственно. Эти классы разделят общую структуру столбцов в соотношении 3: 6: 3.

Таким образом, навигация займет первые 3 столбца, область основного контента займет следующие 6 столбцов, а остальное содержимое займет последние 3 столбца.

4. Добавление навигации

Bootstrap предоставляет компонент Navbar для отображения элемента навигации. Компонент навигационной панели Bootstrap содержит кнопку-переключатель (3 маленькие горизонтальные линии справа) для отображения и скрытия элементов навигации.

Добавьте этот код ниже под заголовком раздела: 

    
<nav class="navbar navbar-expand-lg navbar-light bg-light">
  <a class="navbar-brand" href="#">Navbar</a>
  <button class="navbar-toggler" type="button" data-toggle="collapse"data-target="#navbarNav"
        aria-controls="navbarNav" aria-expanded="false" aria-label="Toggle navigation">
    <span class="navbar-toggler-icon"></span>
  </button>
  <div class="collapse navbar-collapse" id="navbarNav">
    <ul class="navbar-nav">
      <li class="nav-item active">
        <a class="nav-link" href="#">Домой<span class="sr-only">(current)</span></a>
      </li>
      <li class="nav-item">
        <a class="nav-link" href="#">О нас</a>
      </li>
      <li class="nav-item">
        <a class="nav-link" href="#">Продукты</a>
      </li>
      <li class="nav-item">
        <a class="nav-link" href="#">Контакты</a>
      </li>
    </ul>
  </div>
</nav>

See the Pen gOaWWWO by Julia Kalinichenko (@yuliakalina) on CodePen.

В данном примере, использовано несколько классов для .navbar

  • .navbar-toggler для использования с нашим плагином коллапса и другими способами переключения навигации.
  • Навбарам требуется перенос .navbar с .navbar-expand {-sm | -md | -lg | -xl} для адаптивных классов свертывания и цветовой схемы.
  • Элемент </div> с .navbar-collapse состоит из ссылок и элемента управления формы для поиска.

4. Добавление Glyphicons

Glyphicon - это библиотека значков, которая использует символы Юникода для отображения значка. Glyphicon предоставляют разные классы для представления разных значков. Все glyphicon являются платными, но с помощью Bootstrap мы можем использовать некоторые из них бесплатно.

Из соображений производительности для всех значков требуется базовый класс и отдельный класс значков.Чтобы создать иконку, нам просто нужно добавить элемент </span> и предоставить 2 класса. Одним из них является .glyphicon, а другой - класс .glyphicon- *icon.

Чтобы использовать, поместите следующий код в любом месте. Не забудьте оставить пробел между значком и текстом для правильного заполнения. 

    
<span class="glyphicon glyphicon-home" aria-hidden="true"></span>

Приведенная выше строка создаст значок дома. Полный список классов значков доступен здесь.

Используйте их в кнопках, группах кнопок для панели инструментов, навигации или ввода с префиксной формой. 

    
<button type="button" class="btn btn-default btn-lg">
  <span class="glyphicon glyphicon-book" aria-hidden="true"></span> Book
</button>

5. Добавление панели

Панели являются элементами контейнера и используются для хранения другого содержимого. Иногда нам нужно отобразить некоторый контент внутри окна, тогда мы можем использовать панели. Bootstrap предоставляет отзывчивые панели.

Панель состоит из 3 частей

  • Заголовок - определение заголовка панели.
  • Тело - содержит основной контент.
  • Подвал (footer) 
    
<div class="container">
  <h2>Панели с классом панели по умолчанию</h2>
  <div class="panel-group">
    <div class="panel panel-default">
      <div class="panel-heading">Панель с классом панели по умолчанию</div>
      <div class="panel-heading">Панель Заголовок</div>
      <div class="panel-body">Панель Контент</div>
    </div>
      <div class="panel-footer">Панель Подвал</div>
<div class="panel panel-primary">


      <div class="panel-heading">Панель главного класса</div>
      <div class="panel-body">Панель Контент</div>
    </div>
  • Используйте класс .panel-heading для создания заголовка панели.
  • Используйте класс .panel-body для создания тела панели.
  • Используйте класс .panel-footer для создания панели нижнего колонтитула.
  • Чтобы сгруппировать много панелей вместе, оберните </div>классом .panel-group вокруг них.

Чтобы раскрасить панель, используйте контекстные классы .panel-default, .panel-primary, .panel-success, .panel-info, .panel-warning или .panel-danger

5. Добавление Jumbotron

Jumbotron используется для отображения ведущего текста на веб-странице. Мы можем использовать jumbotron, чтобы выделить некоторую информацию на странице. Bootstrap предоставляет класс .jumbotron для создания компонента jumbotron. 

    
  <div class="jumbotron">
     <h1 class="display-4">Bootstrap</h1>
        <p class="lead">
            Bootstrap - это фреймворк HTML, CSS и JavaScript для разработки
            адаптивных мобильных веб-сайтов.
            Bootstrap делает разработку веб-интерфейса быстрее и проще.
            Это сделано для людей всех уровней квалификации,
            устройств всех форм и проектов всех размеров.
        </p>
    <p class="lead">
    <a class="btn btn-primary btn-dark" href="#" role="button">Узнать больше</a>
  </p>
</div>
