1. Добавление отзывчивого изображения

Bootstrap предоставляет класс .img-fluid для создания адаптивных изображений. Этот класс делает их адаптивными для устройств разных размеров.

Bootstrap также предоставляет другие классы для изображений.

  • .rounded-circle - формирует изображение в круг.
  • .rounded float-left/right - выравнивает изображение.
  • .img-thumbnail - придает изображению округлую границу в 1 пиксель.
    
<!-- .img-fluid-->
<img src="#" class="img-fluid" alt="image">

<!-- .rounded-circle-->
<img src="#" class="rounded-circle" alt="image">

<!-- .float-left/right-->
<img src="#" class="float-left">
<img src="#" class="float-right">

<!-- .img-thumbnail-->
<img src="#" class="img-thumbnail" alt="image">

See the Pen ZEbKydz by Julia Kalinichenko (@yuliakalina) on CodePen.

2. Добавление отзывчивой таблицы

Таблицы используются для отображения структурированных данных в виде строк и столбцов. Boostrap предоставляет класс .table для создания адаптивной таблицы. Класс .table определяет расположение таблицы и ставит границу.

Bootstrap также предоставляет другие классы для таблиц.

  • .table-responsive{-sm-md-lg-xl} - создает адаптивную таблицу.
  • .table-striped - добавляет "зебру" в таблицу.
  • .table-hover - добавляет эффект наведения (серый цвет фона) на строки таблицы.
  • .table-dark - добавляет в таблицу черный фон.
  • .table-borderless - удаляет границы из таблицы.
    
<div class="container">
  <h2>Таблица</h2>
<div class="table-responsive">
  <table class="table table-bordered table-hover">
    <thead class="thead-dark">
      <tr>
        <th>Имя</th>
        <th>Фамилия</th>
        <th>Email</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td>Юлия</td>
        <td>Калиниченко</td>
        <td>yuliia.kalinichenko@nure.ua</td>
      </tr>
      <tr>
        <td>Юлия</td>
        <td>Калиниченко</td>
        <td>yuliia.kalinichenko@nure.ua</td>
      </tr>
      <tr>
        <td>Юлия</td>
        <td>Калиниченко</td>
        <td>yuliia.kalinichenko@nure.ua</td>
      </tr>
    </tbody>
  </table>
</div>

3. Добавление отзывчивого видео

Bootstrap предоставляет классы, чтобы сделать вставки видео или слайд-шоу на основе ширины родительского элемента, создавая внутреннее соотношение, которое масштабируется на любом устройстве.

Иногда нам нужно отображать сторонние приложения на нашей веб-странице. Таким образом, мы можем использовать тег <embed> , <iframe>, <video> или <object> для встраивания этих сторонних приложений.ние таблицы и ставит границу.

Bootstrap предоставляет класс .embed-responsive для создания адаптивного встраивания. Мы также можем установить соотношение сторон.

  • .embed-responseive-16by9 - Отображает код для вставки или видео в соотношении ширины и высоты 16:9.
  • .embed-responseive-4by3 - Показывать код для вставки или видео в соотношении ширины и высоты 4:3 
    
  <div class="embed-responsive embed-responsive-21by9">
  <iframe class="embed-responsive-item"src="https://www.youtube.com/embed/2JMMnNOhDoc"
   frameborder="0" allow="accelerometer; autoplay; encrypted-media; gyroscope;
   picture-in-picture"allowfullscreen></iframe>
</div>

Таким образом, у нас получилось встроенное видео с YouTube для компьютеров, планшетов и мобильных устройств.
